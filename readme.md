## vim

Installation:

>git clone git://github.com/ryanthomastech/dotfiles ~

Create symlinks:

>ln -s ~/.vim/vimrc ~/.vimrc
>ln -s ~/.vim/gvimrc ~/.gvimrc

Update submodules

>git submodule init
>git submodule update
